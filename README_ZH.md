# DDL管家
[英文版](README.md)
## 介绍
**DDL管家** 是一个帮助用户利用AI有效管理DDL（Deadline，截止日期）的APP。该应用基于安卓系统，用户输入DDL内容和具体日期，由AI拆分任务并生成一个可视化的时间表，帮助用户更好地规划和完成任务。
## 核心功能
1. **DDL拆分与管理** - 利用AI将DDL拆分成更小而具体的任务，帮助用户更好规划时间并缓解焦虑
2. **动态更新任务状态** - 用户可以实时更新任务的完成情况，APP会根据当前完成状态动态调整剩余任务的优先级和时间分配，确保用户能够高效地完成DDL。
3. **智能提醒与个性化建议** - 根据任务的优先级和截止日期，APP会智能地提醒用户即将到期的任务。根据用户个人习惯，APP会给出个性化建议。
4. **可视化时间表** - 生成一个清晰的时间表，帮助用户直观地了解任务的分布和时间安排，提升时间管理效率。

## 技术栈
- Android开发：Kotlin
- AI调用：**[KOOG](https://github.com/JetBrains/koog)** （Kotlin版的Langchain库）

## 快速开始
点击“+”号，输入 DDL 名称、截止日期和具体描述，应用接入大模型解析任务内容，生成合理的子任务细化和日程安排，在应用界面展示当天截止的子任务和相应的 DDL。还可以选择查看所有 DDL 或按照日期查看 DDL。
